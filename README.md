# Personal Portfolio Website

A responsive, multi-page personal portfolio and CV website built from scratch using HTML5, CSS3, and Vanilla JavaScript.

## 🚀 Features
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewports using modern CSS techniques (Flexbox, Grid, and Media Queries).
* **Dynamic Navigation:** Includes a toggleable mobile hamburger menu for better user experience on smaller screens.
* **Client-Side Validation:** The contact form features custom JavaScript validation to ensure required fields are not empty, adhere to character limits, and follow standard email formatting.
* **Semantic Structure:** Developed with clean and accessible HTML5 semantics.

## 🛠️ Technologies Used
* **HTML5**
* **CSS3**
* **JavaScript (Vanilla / ES6+)**

## 📂 Project Structure
* `index.html` - Landing page and social media links.
* `about.html` - Personal biography section.
* `cv.html` - Academic and professional resume layout.
* `projects.html` - Showcase gallery of past projects.
* `contact.html` - Interactive contact form.
* `css/styles.css` - Global stylesheets.
* `js/script.js` - DOM manipulation and form validation logic.
